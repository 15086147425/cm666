JavaScript SDK 使用指南

安装

使用 npm 或 yarn 安装 SDK：

npm install devconnect-sdk
# 或
yarn add devconnect-sdk

初始化

在您的应用中初始化 SDK：

import DevConnect from 'devconnect-sdk';

// 初始化 SDK
const devconnect = new DevConnect({
    apiKey: 'YOUR_API_KEY',
    environment: 'production' // 或 'sandbox'
});

基本用法

调用 API 示例：

// 获取用户信息
async function getUserInfo(userId) {
    try {
        const userInfo = await devconnect.users.get(userId);
        console.log('用户信息:', userInfo);
        return userInfo;
    } catch (error) {
        console.error('获取用户信息失败:', error);
        throw error;
    }
}

// 调用示例
getUserInfo('user_123456');

Python SDK 使用指南

安装

使用 pip 安装 SDK：

pip install devconnect-sdk

初始化

在您的应用中初始化 SDK：

from devconnect import DevConnect

# 初始化 SDK
devconnect = DevConnect(
    api_key='YOUR_API_KEY',
    environment='production'  # 或 'sandbox'
)

基本用法

调用 API 示例：

# 获取用户信息
def get_user_info(user_id):
    try:
        user_info = devconnect.users.get(user_id)
        print('用户信息:', user_info)
        return user_info
    except Exception as e:
        print('获取用户信息失败:', e)
        raise

# 调用示例
user_info = get_user_info('user_123456')

Java SDK 使用指南

安装

在 pom.xml 中添加依赖：

<dependency>
    <groupId>com.devconnect</groupId>
    <artifactId>devconnect-sdk</artifactId>
    <version>1.0.0</version>
</dependency>

初始化

在您的应用中初始化 SDK：

import com.devconnect.DevConnect;
import com.devconnect.config.Environment;

// 初始化 SDK
DevConnect devconnect = new DevConnect.Builder()
    .apiKey("YOUR_API_KEY")
    .environment(Environment.PRODUCTION) // 或 Environment.SANDBOX
    .build();

基本用法

调用 API 示例：

// 获取用户信息
public UserInfo getUserInfo(String userId) {
    try {
        UserInfo userInfo = devconnect.users().get(userId);
        System.out.println("用户信息: " + userInfo);
        return userInfo;
    } catch (Exception e) {
        System.err.println("获取用户信息失败: " + e.getMessage());
        throw e;
    }
}

// 调用示例
UserInfo userInfo = getUserInfo("user_123456");

PHP SDK 使用指南

安装

使用 Composer 安装 SDK：

composer require devconnect/sdk

初始化

在您的应用中初始化 SDK：

require_once 'vendor/autoload.php';

use DevConnect\DevConnect;

// 初始化 SDK
$devconnect = new DevConnect([
    'apiKey' => 'YOUR_API_KEY',
    'environment' => 'production' // 或 'sandbox'
]);

基本用法

调用 API 示例：

// 获取用户信息
function getUserInfo($userId) {
    global $devconnect;
    try {
        $userInfo = $devconnect->users->get($userId);
        echo '用户信息: ' . json_encode($userInfo);
        return $userInfo;
    } catch (Exception $e) {
        echo '获取用户信息失败: ' . $e->getMessage();
        throw $e;
    }
}

// 调用示例
$userInfo = getUserInfo('user_123456');

API 调用示例

cURL

curl -X GET "https://api.devconnect.com/v1/users" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

JavaScript (Fetch)

fetch('https://api.devconnect.com/v1/users', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));

Python (Requests)

import requests

url = 'https://api.devconnect.com/v1/users'
headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get(url, headers=headers)
data = response.json()
print(data)

常见问题

解答您在接入过程中可能遇到的问题

注册开发者账号后，登录开发者控制台，在"API 密钥"页面可以创建和管理您的 API 密钥。每个 API 密钥都有不同的权限范围，您可以根据需要创建多个密钥。

是的，我们的 API 有调用频率限制。免费版用户每分钟最多可调用 60 次，标准版用户每分钟最多可调用 600 次，企业版用户每分钟最多可调用 6000 次。您可以在响应头中查看当前的使用情况。

我们的 API 使用标准的 HTTP 状态码来表示请求结果。当遇到错误时，响应体中会包含详细的错误信息和错误代码。建议您在代码中捕获这些错误并进行适当的处理，如重试机制或用户友好的错误提示。

我们的 SDK 支持多种平台，包括 Web（浏览器）、Node.js、Python、Java、PHP、iOS 和 Android。每个 SDK 都经过优化，以适应特定平台的特性和最佳实践。

登录开发者控制台，进入"账单"页面，您可以选择适合您需求的付费计划并完成支付。升级后，您的 API 调用限制和功能将立即更新，无需更改代码。
